# Disable-DscDebug

## 概要
停止對 DSC 資源的偵錯工具。

## SYNTAX

```
Disable-DscDebug [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
**Enable-dscdebug** 指令程式要求 Windows PowerShell DESIRED STATE CONFIGURATION (dsc) 引擎會停止對 dsc 資源的偵測。
如果 DSC 引擎目前不在偵測模式中，此 Cmdlet 就不會有任何作用。

## 範例

### 範例1：停止資源調試

```
PS C:\> Disable-DscDebug
```

此命令指示本機 Configuration Manager (LCM) 的 DSC 引擎停止資源調試。

### 範例2：檢查引擎狀態並停止調試

```
PS C:\> if((Get-DscLocalConfigurationManager).DebugMode -like '*Break*'){Disable-DscDebug}
```

此命令會檢查 DSC 引擎狀態，而且只有在已處於偵測模式時，才會停止資源偵錯工具
