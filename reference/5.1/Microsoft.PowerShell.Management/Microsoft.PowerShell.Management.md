# Microsoft.PowerShell.Management 模組

## 描述

本章節包含與 PowerShell Microsoft. PowerShell. 管理模組一起安裝之 Cmdlet 的說明主題。 管理模組包含可協助您在 PowerShell 中管理 Windows 的 Cmdlet。

## Microsoft. PowerShell. 管理 Cmdlet

### [Add-Computer](Add-Computer.md)
將本機電腦新增到網域或工作群組。

### [Add-Content](Add-Content.md)
新增內容至指定的項目，例如將文字新增到檔案。

### [Checkpoint-Computer](Checkpoint-Computer.md)
在本機電腦上建立系統還原點。

### [Clear-Content](Clear-Content.md)
刪除項目的內容，但不會刪除項目。

### [Clear-EventLog](Clear-EventLog.md)
清除本機或遠端電腦上指定之事件記錄檔中的所有專案。

### [Clear-Item](Clear-Item.md)
清除專案的內容，但不會刪除專案。

### [Clear-ItemProperty](Clear-ItemProperty.md)
清除屬性的值，但不刪除屬性。

### [Clear-RecycleBin](Clear-RecycleBin.md)
清除回收站的內容。

### [Complete-Transaction](Complete-Transaction.md)
認可使用中交易。

### [Convert-Path](Convert-Path.md)
將路徑從 PowerShell 路徑轉換為 PowerShell 提供者路徑。

### [Copy-Item](Copy-Item.md)
將項目從某個位置複製到另一個位置。

### [Copy-ItemProperty](Copy-ItemProperty.md)
從指定的位置將屬性和值複製到另一個位置。

### [Debug-Process](Debug-Process.md)
偵錯本機電腦上執行的一或多個處理程序。

### [Disable-ComputerRestore](Disable-ComputerRestore.md)
在指定的檔案系統磁碟機上停用系統還原功能。

### [Enable-ComputerRestore](Enable-ComputerRestore.md)
在指定的檔案系統磁碟機上啟用系統還原功能。

### [Get-ChildItem](Get-ChildItem.md)
取得一或多個指定之位置中的項目與子項目。

### [Get-Clipboard](Get-Clipboard.md)
取得目前的 Windows 剪貼簿專案。

### [Get-ComputerInfo](Get-ComputerInfo.md)
取得系統和作業系統屬性的合併物件。

### [Get-ComputerRestorePoint](Get-ComputerRestorePoint.md)
在本機電腦上取得還原點。

### [Get-Content](Get-Content.md)
在指定的位置取得項目的內容。

### [Get-ControlPanelItem](Get-ControlPanelItem.md)
取得控制台項目。

### [Get-EventLog](Get-EventLog.md)
取得本機電腦或遠端電腦上事件記錄檔中的事件，或事件記錄檔的清單。

### [Get-HotFix](Get-HotFix.md)
取得安裝在本機或遠端電腦上的修補程式。

### [Get-Item](Get-Item.md)
取得指定位置的項目。

### [Get-ItemProperty](Get-ItemProperty.md)
取得指定項目的屬性。

### [Get-ItemPropertyValue](Get-ItemPropertyValue.md)
取得指定項目的一或多個屬性的值。

### [Get-Location](Get-Location.md)
取得目前工作位置或位置堆疊的相關資訊。

### [Get-Process](Get-Process.md)
取得在本機電腦或遠端電腦上執行的處理程序。

### [Get-PSDrive](Get-PSDrive.md)
取得目前工作階段中的磁碟機。

### [Get-PSProvider](Get-PSProvider.md)
取得指定 PowerShell 提供者的相關資訊。

### [Get-Service](Get-Service.md)
取得本機電腦或遠端電腦上的服務。

### [Get-TimeZone](Get-TimeZone.md)
取得目前的時區或可用時區的清單。

### [Get-Transaction](Get-Transaction.md)
取得目前的 (使用中) 交易。

### [Get-WmiObject](Get-WmiObject.md)
取得 WMI 類別的實例或可用類別的相關資訊。

### [Invoke-Item](Invoke-Item.md)
在指定項目上執行預設動作。

### [Invoke-WmiMethod](Invoke-WmiMethod.md)
呼叫 WMI 方法。

### [Join-Path](Join-Path.md)
將路徑和子路徑結合成單一路徑。

### [Limit-EventLog](Limit-EventLog.md)
設定限制事件記錄檔大小和其項目存留期的事件記錄檔屬性。

### [Move-Item](Move-Item.md)
將項目從某個位置移動到另一個位置。

### [Move-ItemProperty](Move-ItemProperty.md)
將屬性從某個位置移動到另一個位置。

### [New-EventLog](New-EventLog.md)
在本機或遠端電腦上建立新的事件記錄檔和新的事件來源。

### [New-Item](New-Item.md)
建立新的項目。

### [New-ItemProperty](New-ItemProperty.md)
建立項目的新屬性，並設定其值。

### [New-PSDrive](New-PSDrive.md)
建立暫存和持續連線的網路磁碟機。

### [New-Service](New-Service.md)
建立新的 Windows 服務。

### [New-WebServiceProxy](New-WebServiceProxy.md)
建立 Web 服務 proxy 物件，可讓您使用和管理 PowerShell 中的 Web 服務。

### [Pop-Location](Pop-Location.md)
將目前的位置變更至最近推送到堆疊上的位置。

### [Push-Location](Push-Location.md)
在位置堆疊頂端新增目前的位置。

### [Register-WmiEvent](Register-WmiEvent.md)
訂閱 Windows Management Instrumentation (WMI) 事件。

### [Remove-Computer](Remove-Computer.md)
將本機電腦從其網域中移除。

### [Remove-EventLog](Remove-EventLog.md)
刪除事件記錄檔或取消註冊事件來源。

### [Remove-Item](Remove-Item.md)
刪除指定的項目。

### [Remove-ItemProperty](Remove-ItemProperty.md)
刪除項目的屬性及其值。

### [Remove-PSDrive](Remove-PSDrive.md)
刪除暫存的 PowerShell 磁片磁碟機，並中斷對應網路磁碟機機的連線。

### [Remove-WmiObject](Remove-WmiObject.md)
刪除現有 Windows Management Instrumentation (WMI) 類別的執行個體。

### [Rename-Computer](Rename-Computer.md)
重新命名電腦。

### [Rename-Item](Rename-Item.md)
重新命名 PowerShell 提供者命名空間中的專案。

### [Rename-ItemProperty](Rename-ItemProperty.md)
重新命名項目的屬性。

### [Reset-ComputerMachinePassword](Reset-ComputerMachinePassword.md)
重設電腦的電腦帳戶密碼。

### [Resolve-Path](Resolve-Path.md)
解析路徑中的萬用字元，並顯示路徑內容。

### [Restart-Computer](Restart-Computer.md)
重新開機本機電腦和遠端電腦上的作業系統。

### [Restart-Service](Restart-Service.md)
停止然後啟動一或多個服務。

### [Restore-Computer](Restore-Computer.md)
在本機電腦上啟動系統還原。

### [Resume-Service](Resume-Service.md)
繼續一或多個擱置 (暫停) 的服務。

### [Set-Clipboard](Set-Clipboard.md)
設定目前的 Windows 剪貼簿項目。

### [Set-Content](Set-Content.md)
寫入新的內容，或取代檔案中的現有內容。

### [Set-Item](Set-Item.md)
將項目的值變更為命令中指定的值。

### [Set-ItemProperty](Set-ItemProperty.md)
建立或變更項目屬性的值。

### [Set-Location](Set-Location.md)
將目前的工作位置設定為指定的位置。

### [Set-Service](Set-Service.md)
啟動、停止並擱置服務，然後變更其屬性。

### [Set-TimeZone](Set-TimeZone.md)

### [Set-WmiInstance](Set-WmiInstance.md)

### [Show-ControlPanelItem](Show-ControlPanelItem.md)

### [Show-EventLog](Show-EventLog.md)
在 [事件檢視器] 中顯示本機或遠端電腦的事件記錄檔。

### [Split-Path](Split-Path.md)
傳回路徑的指定部分。

### [Start-Process](Start-Process.md)
啟動本機電腦上的一或多個處理程序。

### [Start-Service](Start-Service.md)

### [Start-Transaction](Start-Transaction.md)

### [Stop-Computer](Stop-Computer.md)

### [Stop-Process](Stop-Process.md)

### [Stop-Service](Stop-Service.md)

### [Suspend-Service](Suspend-Service.md)

### [Test-ComputerSecureChannel](Test-ComputerSecureChannel.md)
