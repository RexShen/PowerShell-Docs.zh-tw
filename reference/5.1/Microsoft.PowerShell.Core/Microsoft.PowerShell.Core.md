# Microsoft.PowerShell.Core 模組

## 描述

本章節包含與 PowerShell Microsoft. 核心模組一起安裝之 Cmdlet 的說明主題。 核心模組包含可管理 PowerShell 基本功能的 Cmdlet 和提供者。

## Microsoft PowerShell Cmdlet

### [Add-History](Add-History.md)
將項目附加至工作階段歷程記錄。

### [Add-PSSnapin](Add-PSSnapin.md)
新增一或多個 Windows PowerShell 嵌入式管理單元至目前的工作階段。

### [Clear-History](Clear-History.md)
從 PowerShell 會話命令歷程記錄中刪除專案。

### [Clear-Host](Clear-Host.md)
清除主機程式中的顯示。

### [Connect-PSSession](Connect-PSSession.md)
重新連線至已中斷連線的工作階段。

### [Debug-Job](Debug-Job.md)
對執行中的背景、遠端或 Windows PowerShell 工作流程工作進行偵錯。

### [Disable-PSRemoting](Disable-PSRemoting.md)
防止 PowerShell 端點接收遠端連線。

### [Disable-PSSessionConfiguration](Disable-PSSessionConfiguration.md)
停用本機電腦上的工作階段設定。

### [Disconnect-PSSession](Disconnect-PSSession.md)
從工作階段中斷連線。

### [Enable-PSRemoting](Enable-PSRemoting.md)
設定電腦接收遠端命令。

### [Enable-PSSessionConfiguration](Enable-PSSessionConfiguration.md)
啟用本機電腦上的工作階段設定。

### [Enter-PSHostProcess](Enter-PSHostProcess.md)
連線到互動式工作階段並進入具本機處理程序的互動式工作階段。

### [Enter-PSSession](Enter-PSSession.md)
啟動遠端電腦的互動式工作階段。

### [Exit-PSHostProcess](Exit-PSHostProcess.md)
關閉具有本機進程的互動式會話。

### [Exit-PSSession](Exit-PSSession.md)
結束遠端電腦的互動式工作階段。

### [Export-Console](Export-Console.md)
將目前工作階段中的嵌入式管理單元名稱匯出到主控台檔案。

### [Export-ModuleMember](Export-ModuleMember.md)
指定要匯出的模組成員。

### [ForEach-Object](ForEach-Object.md)
針對輸入物件集合中的每個項目執行操作。

### [Get-Command](Get-Command.md)
取得所有命令。

### [Get-Help](Get-Help.md)
顯示 PowerShell 命令和概念的相關資訊。

### [Get-History](Get-History.md)
取得目前工作階段期間輸入的命令清單。

### [Get-Job](Get-Job.md)
取得目前會話中正在執行的 PowerShell 背景工作。

### [Get-Module](Get-Module.md)
取得已匯入或可匯入目前工作階段的模組。

### [Get-PSHostProcessInfo](Get-PSHostProcessInfo.md)
取得 PowerShell 主機的相關處理資訊。

### [Get-PSSession](Get-PSSession.md)
取得本機和遠端電腦上的 PowerShell 會話。

### [Get-PSSessionCapability](Get-PSSessionCapability.md)
取得特定使用者對於限制工作階段設定的能力。

### [Get-PSSessionConfiguration](Get-PSSessionConfiguration.md)
取得電腦上已登錄的工作階段設定。

### [Get-PSSnapin](Get-PSSnapin.md)
取得電腦上的 Windows PowerShell 嵌入式管理單元。

### [Get-Verb](Get-Verb.md)
取得核准的 PowerShell 動詞。

### [Import-Module](Import-Module.md)
將模組新增到目前的工作階段。

### [Invoke-Command](Invoke-Command.md)
在本機與遠端電腦上執行命令。

### [Invoke-History](Invoke-History.md)
從工作階段歷程記錄執行命令。

### [New-Module](New-Module.md)
建立只存在記憶體中的新動態模組。

### [New-ModuleManifest](New-ModuleManifest.md)
建立新的模組資訊清單。

### [New-PSRoleCapabilityFile](New-PSRoleCapabilityFile.md)
建立檔案，該檔案會定義要透過會話設定公開的一組功能。

### [New-PSSession](New-PSSession.md)
建立連到本機或遠端電腦的持續連線。

### [New-PSSessionConfigurationFile](New-PSSessionConfigurationFile.md)
建立定義工作階段設定的檔案。

### [New-PSSessionOption](New-PSSessionOption.md)
建立包含 PSSession 之進階選項的物件。

### [New-PSTransportOption](New-PSTransportOption.md)
建立包含工作階段設定之進階選項的物件。

### [Out-Default](Out-Default.md)
將輸出傳送到預設的格式器和預設的輸出 Cmdlet。

### [Out-Host](Out-Host.md)
將輸出傳送至命令列。

### [Out-Null](Out-Null.md)
隱藏輸出，而不是將它向下傳送或顯示。

### [Receive-Job](Receive-Job.md)
取得目前會話中 PowerShell 背景工作的結果。

### [Receive-PSSession](Receive-PSSession.md)
取得已中斷連線會話中的命令結果。

### [Register-ArgumentCompleter](Register-ArgumentCompleter.md)
註冊自訂引數完成器。

### [Register-PSSessionConfiguration](Register-PSSessionConfiguration.md)
建立並登錄新的工作階段設定。

### [Remove-Job](Remove-Job.md)
刪除 PowerShell 背景工作。

### [Remove-Module](Remove-Module.md)
移除目前工作階段的模組。

### [Remove-PSSession](Remove-PSSession.md)
 (Pssession) 關閉一或多個 PowerShell 會話。

### [Remove-PSSnapin](Remove-PSSnapin.md)
從目前的工作階段中移除 Windows PowerShell 嵌入式管理單元。

### [Resume-Job](Resume-Job.md)
重新啟動暫停的工作。

### [Save-Help](Save-Help.md)
下載最新的說明檔並儲存至檔案系統目錄。

### [Set-PSDebug](Set-PSDebug.md)
開啟和關閉指令碼偵錯功能、設定追蹤層級，以及切換 strict 模式。

### [Set-PSSessionConfiguration](Set-PSSessionConfiguration.md)
變更已登錄的工作階段設定屬性。

### [Set-StrictMode](Set-StrictMode.md)
在運算式、指令碼和指令碼區塊中建立並強制執行編碼規則。

### [Start-Job](Start-Job.md)
啟動 PowerShell 背景工作。

### [Stop-Job](Stop-Job.md)
停止 PowerShell 背景工作。

### [Suspend-Job](Suspend-Job.md)
暫時停止工作流程工作。

### [Test-ModuleManifest](Test-ModuleManifest.md)
驗證模組資訊清單檔案可精確地說明模組的內容。

### [Test-PSSessionConfigurationFile](Test-PSSessionConfigurationFile.md)
驗證工作階段設定檔中的索引鍵和值。

### [Unregister-PSSessionConfiguration](Unregister-PSSessionConfiguration.md)
從電腦刪除已註冊的工作階段設定。

### [Update-Help](Update-Help.md)
下載最新的說明檔並安裝於電腦上。

### [Wait-Job](Wait-Job.md)
抑制命令提示字元，直到會話中正在執行的其中一個或所有 PowerShell 背景工作完成為止。

### [Where-Object](Where-Object.md)
根據屬性值從集合中選取物件。
