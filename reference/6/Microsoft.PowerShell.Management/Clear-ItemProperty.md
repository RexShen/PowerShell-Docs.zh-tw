# Clear-ItemProperty

## 概要
清除屬性的值，但不刪除屬性。

## SYNTAX

### 路徑 (預設值)

```
Clear-ItemProperty [-Path] <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### LiteralPath

```
Clear-ItemProperty -LiteralPath <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

`Clear-ItemProperty`Cmdlet 會清除屬性的值，但不會刪除屬性。
您可以使用此 Cmdlet 刪除登錄值中的資料。

## 範例

### 範例1：清除登錄機碼的值

此命令會清除的 "MyApp" 子機碼中 "Options" 登錄值的資料 `HKEY_LOCAL_MACHINE\Software\MyCompany` 。

```powershell
Clear-ItemProperty -Path "HKLM:\Software\MyCompany\MyApp" -Name "Options"
```

## 注意

- 您可以使用 `Clear-ItemProperty` 刪除登錄值中的資料而不刪除值。
  如果值的資料類型是二進位或 DWORD，清除資料會將值設定為零。
  否則，值會是空的。
- 此 `Clear-ItemProperty` Cmdlet 是針對與任何提供者公開的資料搭配使用所設計。 若要列出工作階段中可用的提供者，請輸入 `Get-PSProvider`。 如需詳細資訊，請參閱 [about_Providers](../Microsoft.PowerShell.Core/About/about_Providers.md)。
