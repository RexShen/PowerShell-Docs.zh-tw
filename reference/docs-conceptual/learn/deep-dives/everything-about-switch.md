## <a name="switch-statement"></a><span data-ttu-id="0b18c-117">Switch 陳述式</span><span class="sxs-lookup"><span data-stu-id="0b18c-117">Switch statement</span></span>

'Wednesday'
```

<span data-ttu-id="0b18c-148">我們可以使用 `-wildcard` switch 來啟用萬用字元支援。</span><span class="sxs-lookup"><span data-stu-id="0b18c-148">We can enable wildcard support with the `-wildcard` switch.</span></span> <span data-ttu-id="0b18c-149">其會使用與 `-like` 運算子相同的萬用字元邏輯來進行每個比對。</span><span class="sxs-lookup"><span data-stu-id="0b18c-149">This uses the same wildcard logic as the `-like` operator to do each match.</span></span>

    }
    default
    {
        Write-Information $message
    }
}
```

```Output

```
