<span data-ttu-id="f971b-185">在 PowerShell 7 之前，您必須有自訂程式碼，才能載入適當的原生 dll，以便受控程式庫正確地找到它。</span><span class="sxs-lookup"><span data-stu-id="f971b-185">Prior to PowerShell 7, one would have to have custom code to load the appropriate native dll so that the managed library can find it correctly.</span></span>
