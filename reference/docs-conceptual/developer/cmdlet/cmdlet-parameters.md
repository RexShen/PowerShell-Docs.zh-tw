# <a name="cmdlet-parameters"></a>Cmdlet 參數

Cmdlet 參數提供允許 Cmdlet 接受輸入的機制。 參數可以直接從命令列接受輸入，或透過管線傳遞給 Cmdlet 的物件，這些參數 (也稱為 *值*) 這些參數可以指定 Cmdlet 接受的輸入、Cmdlet 應如何執行其動作，以及 Cmdlet 傳回給管線的資料。
