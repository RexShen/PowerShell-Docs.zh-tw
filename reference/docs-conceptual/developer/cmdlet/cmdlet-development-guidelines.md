# <a name="cmdlet-development-guidelines"></a>Cmdlet 開發指導方針

本節中的主題提供開發指導方針，您可以使用這些指導方針來產生格式正確的 Cmdlet。 藉由利用 Windows PowerShell 執行時間所提供的一般功能，並遵循這些指導方針，您可以最簡單的方式開發健全的 Cmdlet，並為使用者提供一致的體驗。 此外，您將會降低測試負擔，因為一般功能不需要重新測試。
