# <a name="how-to-create-a-console-shell"></a><span data-ttu-id="311eb-102">如何建立主控台殼層</span><span class="sxs-lookup"><span data-stu-id="311eb-102">How to Create a Console Shell</span></span>

<span data-ttu-id="311eb-103">Windows PowerShell 提供一個 Shell 工具（也稱為「建立套件」），可用來建立不可延伸的主控台 Shell。</span><span class="sxs-lookup"><span data-stu-id="311eb-103">Windows PowerShell provides a Make-Shell tool, also referred to as the "make-kit", that is used to create a console shell that is not extensible.</span></span> <span data-ttu-id="311eb-104">使用這項新工具所建立的 shell 之後，將無法透過 Windows PowerShell 嵌入式管理單元延伸。</span><span class="sxs-lookup"><span data-stu-id="311eb-104">Shells created with this new tool cannot be extended later through a Windows PowerShell snap-in.</span></span>

## <a name="syntax"></a><span data-ttu-id="311eb-105">語法</span><span class="sxs-lookup"><span data-stu-id="311eb-105">Syntax</span></span>

<span data-ttu-id="311eb-106">以下是用來從檔案內執行 Make Shell 的語法。</span><span class="sxs-lookup"><span data-stu-id="311eb-106">Here is the syntax used to run Make-Shell from within a make-file.</span></span>

```
make-shell
  -out n.exe
  -namespace ns
  [ -lib libdirectory1[,libdirectory2,..] ]
  [ -reference ca1.dll[,ca2.dll,...] ]
  [ -formatdata fd1.format.ps1xml[,fd2.format.ps1xml,...] ]
  [ -typedata td1.type.ps1xml[,td2.type.ps1xml,...] ]
  [ -source c1.cs [,c2.cs,...] ]
  [ -authorizationmanager authorizationManagerType ]
  [ -win32icon i.ico ]
  [ -initscript p.ps1 ]
  [ -builtinscript s1.ps1[,s2.ps1,...] ]
  [ -resource resourcefile.txt ]
  [ -cscflags cscFlags ]
  [ -? | -help ]
```
