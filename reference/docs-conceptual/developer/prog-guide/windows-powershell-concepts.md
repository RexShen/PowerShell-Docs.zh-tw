|主題名稱|說明|
|----------------|-----------------|
|[about_Objects](/powershell/module/microsoft.powershell.core/about/about_objects)|PowerShell 物件的描述。 如需詳細資訊，請參閱 [關於建立物件](/powershell/module/microsoft.powershell.core/about/about_object_creation)|
