## <a name="where-selection-sets-are-defined-and-referenced"></a><span data-ttu-id="c72c5-107">定義和參考選擇集的位置</span><span class="sxs-lookup"><span data-stu-id="c72c5-107">Where Selection Sets are Defined and Referenced</span></span>

<span data-ttu-id="c72c5-108">您可以將選取範圍定義為可供格式檔案中定義的所有視圖和控制項使用的一般資料的一部分。</span><span class="sxs-lookup"><span data-stu-id="c72c5-108">You define selection sets as part of the common data that can be used by all the views and controls defined in the formatting file.</span></span> <span data-ttu-id="c72c5-109">下列範例顯示如何定義三個選取範圍。</span><span class="sxs-lookup"><span data-stu-id="c72c5-109">The following example shows how to define three selection sets.</span></span>

```xml
<Configuration>
  <SelectionSets>
    <SelectionSet>...</SelectionSet>
    <SelectionSet>...</SelectionSet>
    <SelectionSet>...</SelectionSet>
  </SelectionSets>
</Configuration>
```

<span data-ttu-id="c72c5-110">您可以透過下列方式參考選擇集：</span><span class="sxs-lookup"><span data-stu-id="c72c5-110">You can reference a selection sets in the following ways:</span></span>

- <span data-ttu-id="c72c5-111">每個 view 都有一個專案 `ViewSelectedBy` ，用來定義要使用 view 來顯示的物件。</span><span class="sxs-lookup"><span data-stu-id="c72c5-111">Each view has a `ViewSelectedBy` element that defines which objects are displayed by using the view.</span></span> <span data-ttu-id="c72c5-112">`ViewSelectedBy`元素具有 `SelectionSetName` 子專案，可指定所有視圖定義所使用的選擇集。</span><span class="sxs-lookup"><span data-stu-id="c72c5-112">The `ViewSelectedBy` element has a `SelectionSetName` child element that specifies the selection set that all the definitions of the view use.</span></span> <span data-ttu-id="c72c5-113">您可以從視圖參考的選擇集數目沒有限制。</span><span class="sxs-lookup"><span data-stu-id="c72c5-113">There is no restriction on the number of selection sets that you can reference from a view.</span></span>

- <span data-ttu-id="c72c5-114">在視圖或控制項的每個定義中， `EntrySelectedBy` 元素會使用該定義來定義要顯示的物件。</span><span class="sxs-lookup"><span data-stu-id="c72c5-114">In each definition of a view or control, the `EntrySelectedBy` element defines which objects are displayed by using that definition.</span></span> <span data-ttu-id="c72c5-115">通常，視圖或控制項只有一個定義，因此物件是由元素所定義 `ViewSelectedBy` 。</span><span class="sxs-lookup"><span data-stu-id="c72c5-115">Typically a view or control has only one definition so the objects are defined by the `ViewSelectedBy` element.</span></span> <span data-ttu-id="c72c5-116">`EntrySelectedBy`定義的元素具有 `SelectionSetName` 指定選取集的子項目。</span><span class="sxs-lookup"><span data-stu-id="c72c5-116">The `EntrySelectedBy` element of the definition has a `SelectionSetName` child element that specifies the selection set.</span></span> <span data-ttu-id="c72c5-117">如果您指定定義的選取範圍，就不能指定元素的任何其他子項目 `EntrySelectedBy` 。</span><span class="sxs-lookup"><span data-stu-id="c72c5-117">If you specify the selection set for a definition, you cannot specify any of the other child elements of the `EntrySelectedBy` element.</span></span>

- <span data-ttu-id="c72c5-118">在視圖或控制項的每個定義中，專案 `SelectionCondition` 可以用來指定使用定義時的條件。</span><span class="sxs-lookup"><span data-stu-id="c72c5-118">In each definition of a view or control, the `SelectionCondition` element can be used to specify a condition for when the definition is used.</span></span> <span data-ttu-id="c72c5-119">`SelectionCondition`元素具有 `SelectionSetName` 子專案，可指定觸發條件的選擇集。</span><span class="sxs-lookup"><span data-stu-id="c72c5-119">The `SelectionCondition` element has a `SelectionSetName` child element that specifies the selection set that triggers the condition.</span></span> <span data-ttu-id="c72c5-120">當選取範圍中定義的任何物件顯示時，就會觸發此條件。</span><span class="sxs-lookup"><span data-stu-id="c72c5-120">The condition is triggered when any of the objects defined in the selection set are displayed.</span></span> <span data-ttu-id="c72c5-121">如需如何設定這些條件的詳細資訊，請參閱[定義資料顯示時的條件](./defining-conditions-for-displaying-data.md)。</span><span class="sxs-lookup"><span data-stu-id="c72c5-121">For more information about how to set these conditions, see [Defining Conditions for when Data is Displayed](./defining-conditions-for-displaying-data.md).</span></span>

## <a name="selection-set-example"></a><span data-ttu-id="c72c5-122">選擇集範例</span><span class="sxs-lookup"><span data-stu-id="c72c5-122">Selection Set Example</span></span>

<span data-ttu-id="c72c5-123">下列範例顯示直接從 Windows PowerShell 所提供的格式化檔案中取得的選擇集 `FileSystem` 。</span><span class="sxs-lookup"><span data-stu-id="c72c5-123">The following example shows a selection set that is taken directly from the `FileSystem` formatting file provided by Windows PowerShell.</span></span> <span data-ttu-id="c72c5-124">如需其他 Windows PowerShell 格式化檔案的詳細資訊，請參閱[Windows Powershell 格式化](./powershell-formatting-files.md)檔案。</span><span class="sxs-lookup"><span data-stu-id="c72c5-124">For more information about other Windows PowerShell formatting files, see [Windows PowerShell Formatting Files](./powershell-formatting-files.md).</span></span>

```xml
<SelectionSets>
  <SelectionSet>
    <Name>FileSystemTypes</Name>
    <Types>
     <TypeName>System.IO.DirectoryInfo</TypeName>
     <TypeName>System.IO.FileInfo</TypeName>
     <TypeName>Deserialized.System.IO.DirectoryInfo</TypeName>
     <TypeName>Deserialized.System.IO.FileInfo</TypeName>
    </Types>
  </SelectionSet>
</SelectionSets>
```

<span data-ttu-id="c72c5-125">在資料表視圖的元素中，會參考先前的選取專案集 `ViewSelectedBy` 。</span><span class="sxs-lookup"><span data-stu-id="c72c5-125">The previous selection set is referenced in the `ViewSelectedBy` element of a table view.</span></span>

```xml
<ViewDefinitions>
  <View>
    <Name>Files</Name>
    <ViewSelectedBy>
      <SelectionSetName>FileSystemTypes</SelectionSetName>
    </ViewSelectedBy>
    <TableControl>...</TableControl>
  </View>
</ViewDefinitions>

```
