## <a name="best-practice-resource-supports--whatif"></a><span data-ttu-id="8cffd-236">最佳做法︰資源支援 -WhatIf</span><span class="sxs-lookup"><span data-stu-id="8cffd-236">Best practice: Resource supports -WhatIf</span></span>
